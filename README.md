# 🛒 E-Commerce Website

A modern and responsive **E-Commerce Frontend Project** built using HTML, CSS, and JavaScript.
This project showcases a clean UI design with multiple pages like Home, Products, Cart, and About.

🔗 **Live Demo:**
👉 [https://rishithak22.github.io/E-COMMERCE/](https://rishithak22.github.io/E-COMMERCE/about.html)

---

## 🚀 Features

* 🏠 **Homepage UI** with product highlights
* 🛍️ **Product Listing Page**
* 🛒 **Dynamic Cart System**

  * Add items to cart
  * Remove items using JavaScript logic
* 📄 **About Page**

  * Brand introduction
  * Core values section (Quality, Trust, Customer Focus)
* 🎯 Clean and structured layout for scalability
* 📱 Fully responsive design

---

## 🧠 Project Highlights

* Implemented **dynamic cart functionality** using JavaScript
* Used **array-based logic (`splice`)** to correctly remove items
* Designed reusable UI components (cards, sections, grids)
* Focused on **real-world e-commerce flow**
